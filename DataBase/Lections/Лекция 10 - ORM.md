**–ù–∞—à —Ñ–æ–∫—É—Å:** –ú—ã –±—É–¥–µ–º —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å ORM –Ω–µ –∫–∞–∫ "–≤–æ–ª—à–µ–±–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ" –Ω–µ—Å—É—â–µ—Å—Ç–≤—É—é—â–µ–π —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–æ–π –ø—Ä–æ–±–ª–µ–º—ã, –∞ –∫–∞–∫ **–ø—Ä–∞–≥–º–∞—Ç–∏—á–Ω—ã–π –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç (–∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω)**, –∫–æ—Ç–æ—Ä—ã–π –ø–æ–º–æ–≥–∞–µ—Ç –∞–≤—Ç–æ–º–∞—Ç–∏–∑–∏—Ä–æ–≤–∞—Ç—å —Ä—É—Ç–∏–Ω–Ω—ã–µ –∑–∞–¥–∞—á–∏ –ø–æ —Ç—Ä–∞–Ω—Å–ª—è—Ü–∏–∏ –º–µ–∂–¥—É —ç—Ç–∏–º–∏ –¥–≤—É–º—è –ø–∞—Ä–∞–¥–∏–≥–º–∞–º–∏, –∏ –∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å, –∫–∞–∫—É—é —Ü–µ–Ω—É –≤ –≤–∏–¥–µ —Å–ª–æ–∂–Ω–æ—Å—Ç–∏ –∏ "–ø—Ä–æ—Ç–µ–∫–∞—é—â–∏—Ö –∞–±—Å—Ç—Ä–∞–∫—Ü–∏–π" –Ω–∞–º –ø—Ä–∏—Ö–æ–¥–∏—Ç—Å—è –∑–∞ —ç—Ç–æ –ø–ª–∞—Ç–∏—Ç—å.

### **–ß–∞—Å—Ç—å 2. Object-Relational Mapper (ORM)**

**ORM** - —ç—Ç–æ —Ç–µ—Ö–Ω–æ–ª–æ–≥–∏—è (–±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –∏–ª–∏ —Ñ—Ä–µ–π–º–≤–æ—Ä–∫), –∫–æ—Ç–æ—Ä–∞—è —Å–æ–∑–¥–∞–µ—Ç "–≤–∏—Ä—Ç—É–∞–ª—å–Ω—É—é –æ–±—ä–µ–∫—Ç–Ω—É—é –±–∞–∑—É –¥–∞–Ω–Ω—ã—Ö" –ø–æ–≤–µ—Ä—Ö —Ä–µ–ª—è—Ü–∏–æ–Ω–Ω–æ–π –°–£–ë–î. –û–Ω–∞ –±–µ—Ä–µ—Ç –Ω–∞ —Å–µ–±—è –≤—Å—é –≥—Ä—è–∑–Ω—É—é —Ä–∞–±–æ—Ç—É –ø–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—é –¥–∞–Ω–Ω—ã—Ö –º–µ–∂–¥—É –¥–≤—É–º—è –º–∏—Ä–∞–º–∏.

**–ò–∑ —á–µ–≥–æ –∫–æ–Ω—Ü–µ–ø—Ç—É–∞–ª—å–Ω–æ —Å–æ—Å—Ç–æ–∏—Ç –ª—é–±–∞—è ORM?**

1.  **–ú–∞–ø–ø–µ—Ä (Mapper):** –ì–ª–∞–≤–Ω–∞—è —á–∞—Å—Ç—å. –≠—Ç–æ –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏—è (–≤ –≤–∏–¥–µ –∫–æ–¥–∞, –¥–µ–∫–æ—Ä–∞—Ç–æ—Ä–æ–≤, XML –∏–ª–∏ YAML), –∫–æ—Ç–æ—Ä–∞—è –æ–ø–∏—Å—ã–≤–∞–µ—Ç, –∫–∞–∫ –∫–ª–∞—Å—Å –∏ –µ–≥–æ –ø–æ–ª—è —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç —Ç–∞–±–ª–∏—Ü–µ –∏ –µ–µ —Å—Ç–æ–ª–±—Ü–∞–º.
2.  **–ú–µ–Ω–µ–¥–∂–µ—Ä —Å—É—â–Ω–æ—Å—Ç–µ–π (Session / Entity Manager / Context):** "–£–º–Ω—ã–π" –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä, –∫–æ—Ç–æ—Ä—ã–π —É–ø—Ä–∞–≤–ª—è–µ—Ç –∂–∏–∑–Ω–µ–Ω–Ω—ã–º —Ü–∏–∫–ª–æ–º –æ–±—ä–µ–∫—Ç–æ–≤. –û–Ω –æ—Ç—Å–ª–µ–∂–∏–≤–∞–µ—Ç, –∫–∞–∫–∏–µ –æ–±—ä–µ–∫—Ç—ã –±—ã–ª–∏ –∑–∞–≥—Ä—É–∂–µ–Ω—ã –∏–∑ –±–∞–∑—ã, –∫–∞–∫–∏–µ –±—ã–ª–∏ –∏–∑–º–µ–Ω–µ–Ω—ã, –∫–∞–∫–∏–µ —Å–æ–∑–¥–∞–Ω—ã –Ω–æ–≤—ã–º–∏, –∞ –∫–∞–∫–∏–µ –ø–æ–º–µ—á–µ–Ω—ã –Ω–∞ —É–¥–∞–ª–µ–Ω–∏–µ.
3.  **–Ø–∑—ã–∫ –∑–∞–ø—Ä–æ—Å–æ–≤ (Query Language):** –í—ã—Å–æ–∫–æ—É—Ä–æ–≤–Ω–µ–≤—ã–π, –æ–±—ä–µ–∫—Ç–Ω–æ-–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ø–æ—Å–æ–± –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è –∑–∞–ø—Ä–æ—Å–æ–≤, –∫–æ—Ç–æ—Ä—ã–π ORM –∑–∞—Ç–µ–º —Ç—Ä–∞–Ω—Å–ª–∏—Ä—É–µ—Ç –≤ SQL. –≠—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å —Ü–µ–ø–æ—á–∫–∞ –º–µ—Ç–æ–¥–æ–≤ (–∫–∞–∫ –≤ Django ORM –∏–ª–∏ SQLAlchemy) –∏–ª–∏ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —è–∑—ã–∫ (–∫–∞–∫ HQL –≤ Hibernate).

```python
# –ü—Ä–∏–º–µ—Ä –Ω–∞ Django ORM
# –ú—ã –ø–∏—à–µ–º –Ω–∞ Python, –¥—É–º–∞–µ–º –æ–± –æ–±—ä–µ–∫—Ç–∞—Ö...
authors_from_london = Author.objects.filter(city="London").order_by("name")

# ... –∞ ORM –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç SQL –∑–∞ –Ω–∞—Å:
# SELECT id, name, city FROM authors WHERE city = 'London' ORDER BY name;
```

#### **–î–≤–∞ –ø—É—Ç–∏: Active Record vs. Data Mapper**

–•–æ—Ç—è –≤—Å–µ ORM —Ä–µ—à–∞—é—Ç –æ–¥–Ω—É –∏ —Ç—É –∂–µ –ø—Ä–æ–±–ª–µ–º—É, –æ–Ω–∏ –¥–µ–ª–∞—é—Ç —ç—Ç–æ, —Å–ª–µ–¥—É—è –¥–≤—É–º —Ä–∞–∑–Ω—ã–º –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–º –ø–∞—Ç—Ç–µ—Ä–Ω–∞–º.

#### **Active Record**
*   **–ò–¥–µ—è:** –û–±—ä–µ–∫—Ç —Å–∞–º –∑–Ω–∞–µ—Ç, –∫–∞–∫ —Å–µ–±—è —Å–æ—Ö—Ä–∞–Ω—è—Ç—å, –æ–±–Ω–æ–≤–ª—è—Ç—å, —É–¥–∞–ª—è—Ç—å –∏ –∑–∞–≥—Ä—É–∂–∞—Ç—å. –ë–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫–∞ –∏ –ª–æ–≥–∏–∫–∞ –ø–µ—Ä—Å–∏—Å—Ç–µ–Ω—Ç–Ω–æ—Å—Ç–∏ (—Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è) —Å–º–µ—à–∞–Ω—ã –≤ –æ–¥–Ω–æ–º –∫–ª–∞—Å—Å–µ.
*   **–ü—Ä–∏–º–µ—Ä—ã:** Ruby on Rails ActiveRecord, Django ORM, Eloquent (Laravel).
*   **–ö–∞–∫ —ç—Ç–æ –≤—ã–≥–ª—è–¥–∏—Ç:**
    ```python
    # Django ORM
    user = User.objects.get(id=1)
    user.name = "New Name"
    user.save() # –û–±—ä–µ–∫—Ç —Å–∞–º —Å–µ–±—è —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç
    user.delete() # –ò —Å–∞–º —Å–µ–±—è —É–¥–∞–ª—è–µ—Ç
    ```
*   **–ü–ª—é—Å—ã:** –ü—Ä–æ—Å—Ç–æ—Ç–∞ –∏ —Å–∫–æ—Ä–æ—Å—Ç—å —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–∏. –û—á–µ–Ω—å –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ –¥–ª—è CRUD-–æ–ø–µ—Ä–∞—Ü–∏–π.
*   **–ú–∏–Ω—É—Å—ã:**
    *   **–ù–∞—Ä—É—à–µ–Ω–∏–µ –ø—Ä–∏–Ω—Ü–∏–ø–∞ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ–π –æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ—Å—Ç–∏ (Single Responsibility Principle):** –ö–ª–∞—Å—Å `User` —Ç–µ–ø–µ—Ä—å –æ—Ç–≤–µ—á–∞–µ—Ç –∏ –∑–∞ –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫—É –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è, –∏ –∑–∞ –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–∏–µ —Å –±–∞–∑–æ–π –¥–∞–Ω–Ω—ã—Ö.
    *   **–°–ª–æ–∂–Ω–æ—Å—Ç—å —Ç–µ—Å—Ç–∏—Ä–æ–≤–∞–Ω–∏—è:** –°–ª–æ–∂–Ω–æ —Ç–µ—Å—Ç–∏—Ä–æ–≤–∞—Ç—å –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫—É –æ–±—ä–µ–∫—Ç–∞ –≤ –∏–∑–æ–ª—è—Ü–∏–∏ –æ—Ç –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö, —Ç–∞–∫ –∫–∞–∫ –≤—ã–∑–æ–≤ `user.save()` —Ç—Ä–µ–±—É–µ—Ç —Ä–µ–∞–ª—å–Ω–æ–≥–æ –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è –∫ –ë–î.

#### **Data Mapper**
*   **–ò–¥–µ—è:** –û–±—ä–µ–∫—Ç - —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö (POCO/POJO). –û–Ω –Ω–∏—á–µ–≥–æ –Ω–µ –∑–Ω–∞–µ—Ç –æ –±–∞–∑–µ –¥–∞–Ω–Ω—ã—Ö. –í—Å—è –ª–æ–≥–∏–∫–∞ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –≤—ã–Ω–µ—Å–µ–Ω–∞ –≤ –æ—Ç–¥–µ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å-–º–∞–ø–ø–µ—Ä (–∏–ª–∏ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π).
*   **–ü—Ä–∏–º–µ—Ä—ã:** Hibernate (Java), Doctrine (PHP), SQLAlchemy (Python).
*   **–ö–∞–∫ —ç—Ç–æ –≤—ã–≥–ª—è–¥–∏—Ç:**
    ```python
    # SQLAlchemy
    user = session.query(User).get(1) # –ó–∞–≥—Ä—É–∂–∞–µ–º —á–µ—Ä–µ–∑ —Å–µ—Å—Å–∏—é
    user.name = "New Name"
    # –û–±—ä–µ–∫—Ç User —Å–∞–º –ø–æ —Å–µ–±–µ –Ω–∏—á–µ–≥–æ –Ω–µ –¥–µ–ª–∞–µ—Ç.
    # –ò–∑–º–µ–Ω–µ–Ω–∏—è –æ—Ç—Å–ª–µ–∂–∏–≤–∞—é—Ç—Å—è –∏ —Å–æ—Ö—Ä–∞–Ω—è—é—Ç—Å—è "—Å–Ω–∞—Ä—É–∂–∏".
    session.commit() # –°–µ—Å—Å–∏—è —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç –≤—Å–µ –æ—Ç—Å–ª–µ–∂–µ–Ω–Ω—ã–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è
    ```
*   **–ü–ª—é—Å—ã:**
    *   **–ß–∏—Å—Ç–∞—è –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞:** –ß–µ—Ç–∫–æ–µ —Ä–∞–∑–¥–µ–ª–µ–Ω–∏–µ –æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ—Å—Ç–∏. –ë–∏–∑–Ω–µ—Å-–º–æ–¥–µ–ª–∏ –Ω–µ "–∑–∞–≥—Ä—è–∑–Ω–µ–Ω—ã" –∫–æ–¥–æ–º –¥–ª—è –ø–µ—Ä—Å–∏—Å—Ç–µ–Ω—Ç–Ω–æ—Å—Ç–∏.
    *   **–í—ã—Å–æ–∫–∞—è —Ç–µ—Å—Ç–∏—Ä—É–µ–º–æ—Å—Ç—å:** –ë–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫—É –º–æ–∂–Ω–æ —Ç–µ—Å—Ç–∏—Ä–æ–≤–∞—Ç—å —Å –ø—Ä–æ—Å—Ç—ã–º–∏ –æ–±—ä–µ–∫—Ç–∞–º–∏ –≤ –ø–∞–º—è—Ç–∏, –±–µ–∑ –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è –∫ –ë–î.
*   **–ú–∏–Ω—É—Å—ã:**
    *   –ë–æ–ª—å—à–µ "–±–æ–π–ª–µ—Ä–ø–ª–µ–π—Ç–∞" (—à–∞–±–ª–æ–Ω–Ω–æ–≥–æ –∫–æ–¥–∞). –¢—Ä–µ–±—É–µ—Ç—Å—è –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞—Ç—å —Å–µ—Å—Å–∏–∏, —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–∏.
    *   –ë–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–π –ø–æ—Ä–æ–≥ –≤—Ö–æ–∂–¥–µ–Ω–∏—è.

#### **Identity Map –∏ Unit of Work**

–î–≤–∞ –≤–∞–∂–Ω–µ–π—à–∏—Ö –ø–∞—Ç—Ç–µ—Ä–Ω–∞, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–µ ORM (–æ—Å–æ–±–µ–Ω–Ω–æ —Ç–∏–ø–∞ Data Mapper), —á—Ç–æ–±—ã —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ —É–ø—Ä–∞–≤–ª—è—Ç—å –æ–±—ä–µ–∫—Ç–∞–º–∏.

*   **Identity Map (–ö–∞—Ä—Ç–∞ –∏–¥–µ–Ω—Ç–∏—á–Ω–æ—Å—Ç–µ–π):**
    *   **–ü—Ä–æ–±–ª–µ–º–∞:** –ö–∞–∫ –∏–∑–±–µ–∂–∞—Ç—å –∑–∞–≥—Ä—É–∑–∫–∏ –¥–≤—É—Ö —Ä–∞–∑–Ω—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏—Ö –æ–¥–Ω—É –∏ —Ç—É –∂–µ —Å—Ç—Ä–æ–∫—É –∏–∑ –±–∞–∑—ã?
    *   **–†–µ—à–µ–Ω–∏–µ:** –°–µ—Å—Å–∏—è ORM –≤–µ–¥–µ—Ç –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π —Å–ª–æ–≤–∞—Ä—å (–∫–∞—Ä—Ç—É), –≥–¥–µ –∫–ª—é—á–æ–º —è–≤–ª—è–µ—Ç—Å—è –ø–µ—Ä–≤–∏—á–Ω—ã–π –∫–ª—é—á —Å—É—â–Ω–æ—Å—Ç–∏, –∞ –∑–Ω–∞—á–µ–Ω–∏–µ–º - —É–∂–µ –∑–∞–≥—Ä—É–∂–µ–Ω–Ω—ã–π –æ–±—ä–µ–∫—Ç. –ö–æ–≥–¥–∞ –≤—ã –∑–∞–ø—Ä–∞—à–∏–≤–∞–µ—Ç–µ `session.get(User, 1)`, —Å–µ—Å—Å–∏—è —Å–Ω–∞—á–∞–ª–∞ —Å–º–æ—Ç—Ä–∏—Ç –≤ —Å–≤–æ—é –∫–∞—Ä—Ç—É. –ï—Å–ª–∏ –æ–±—ä–µ–∫—Ç —Å ID=1 —É–∂–µ —Ç–∞–º, –æ–Ω–∞ –º–≥–Ω–æ–≤–µ–Ω–Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Å—ã–ª–∫—É –Ω–∞ –Ω–µ–≥–æ, –Ω–µ –¥–µ–ª–∞—è –∑–∞–ø—Ä–æ—Å –≤ –ë–î. –ï—Å–ª–∏ –Ω–µ—Ç - –∑–∞–≥—Ä—É–∂–∞–µ—Ç –∏–∑ –ë–î, —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç –≤ –∫–∞—Ä—Ç—É –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç.

3.  **–ü–æ–¥—Ö–æ–¥ –∫ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∑–∞–ø—Ä–æ—Å–æ–≤:**
    *   **–°–≤–æ–π —è–∑—ã–∫ –∑–∞–ø—Ä–æ—Å–æ–≤:** –ù–∞–ø—Ä–∏–º–µ—Ä, HQL (Hibernate), DQL (Doctrine).
    *   **–ò–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –≤ —è–∑—ã–∫ (LINQ-like):** –ó–∞–ø—Ä–æ—Å—ã –ø–∏—à—É—Ç—Å—è —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –Ω–∞—Ç–∏–≤–Ω—ã—Ö –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–π —è–∑—ã–∫–∞.
    *   **–ö–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –∑–∞–ø—Ä–æ—Å–æ–≤ (Query Builder):** –¶–µ–ø–æ—á–∫–∞ –≤—ã–∑–æ–≤–æ–≤ –º–µ—Ç–æ–¥–æ–≤ –¥–ª—è –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è –∑–∞–ø—Ä–æ—Å–∞.

#### **1. Hibernate (Java) / NHibernate (.NET)**

*   **–§–∏–ª–æ—Å–æ—Ñ–∏—è:** "–†–µ–ª—è—Ü–∏–æ–Ω–Ω–∞—è –±–∞–∑–∞ –¥–∞–Ω–Ω—ã—Ö - —ç—Ç–æ –¥–µ—Ç–∞–ª—å —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏. –ú—ã —Å—Ç—Ä–æ–∏–º –æ–±—ä–µ–∫—Ç–Ω—É—é –º–æ–¥–µ–ª—å, –∞ –ø–µ—Ä—Å–∏—Å—Ç–µ–Ω—Ç–Ω–æ—Å—Ç—å - —ç—Ç–æ —Ç–æ, —á—Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–æ–¥ –∫–∞–ø–æ—Ç–æ–º".
*   **–ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω:** **Data Mapper**. –≠—Ç–æ –∫–∞–Ω–æ–Ω–∏—á–µ—Å–∫–∏–π, —Å–∞–º—ã–π –º–æ—â–Ω—ã–π –∏ —Å–ª–æ–∂–Ω—ã–π –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–µ–ª—å —ç—Ç–æ–≥–æ –ø–∞—Ç—Ç–µ—Ä–Ω–∞.
*   **–£—Ä–æ–≤–µ–Ω—å –∞–±—Å—Ç—Ä–∞–∫—Ü–∏–∏:** **–û—á–µ–Ω—å –≤—ã—Å–æ–∫–∏–π ("—Ç—è–∂–µ–ª–∞—è" ORM)**. Hibernate –≤–≤–æ–¥–∏—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Å–≤–æ–∏—Ö –∫–æ–Ω—Ü–µ–ø—Ü–∏–π: `Session`, `SessionFactory`, —Å–æ—Å—Ç–æ—è–Ω–∏—è –æ–±—ä–µ–∫—Ç–æ–≤ (transient, persistent, detached), –Ω–µ—Å–∫–æ–ª—å–∫–æ —É—Ä–æ–≤–Ω–µ–π –∫—ç—à–∏—Ä–æ–≤–∞–Ω–∏—è (L1, L2), —Å–≤–æ–π —è–∑—ã–∫ –∑–∞–ø—Ä–æ—Å–æ–≤ (HQL/JPQL).
*   **–ö–ª—é—á–µ–≤—ã–µ –æ—Ç–ª–∏—á–∏—è:**
    *   **–ú–æ—â–Ω–µ–π—à–µ–µ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ –∂–∏–∑–Ω–µ–Ω–Ω—ã–º —Ü–∏–∫–ª–æ–º:** –ü–∞—Ç—Ç–µ—Ä–Ω—ã `Identity Map` –∏ `Unit of Work` —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω—ã –≤ —Å–∞–º–æ–π –ø–æ–ª–Ω–æ–π –º–µ—Ä–µ.
    *   **–õ–µ–Ω–∏–≤–∞—è –∑–∞–≥—Ä—É–∑–∫–∞ (Lazy Loading):** –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –ø—ã—Ç–∞–µ—Ç—Å—è –±—ã—Ç—å "—É–º–Ω–æ–π" –∏ –∑–∞–≥—Ä—É–∂–∞—Ç—å —Å–≤—è–∑–∞–Ω–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ —Ç–æ–ª—å–∫–æ –ø–æ —Ç—Ä–µ–±–æ–≤–∞–Ω–∏—é, —á—Ç–æ —è–≤–ª—è–µ—Ç—Å—è –∏—Å—Ç–æ—á–Ω–∏–∫–æ–º –∫–∞–∫ —É–¥–æ–±—Å—Ç–≤–∞, —Ç–∞–∫ –∏ –ø—Ä–æ–±–ª–µ–º (N+1).
    *   **–ü—Ä–æ–¥–≤–∏–Ω—É—Ç–æ–µ –∫—ç—à–∏—Ä–æ–≤–∞–Ω–∏–µ:** –ù–∞–ª–∏—á–∏–µ –∫—ç—à–∞ –≤—Ç–æ—Ä–æ–≥–æ —É—Ä–æ–≤–Ω—è (L2 cache) –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ä–∞–∑–¥–µ–ª—è—Ç—å –∑–∞–∫—ç—à–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –º–µ–∂–¥—É —Ä–∞–∑–Ω—ã–º–∏ —Å–µ—Å—Å–∏—è–º–∏ –∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º–∏, —á—Ç–æ —É–Ω–∏–∫–∞–ª—å–Ω–æ –¥–ª—è ORM —Ç–∞–∫–æ–≥–æ –∫–ª–∞—Å—Å–∞.
    *   **–°–ª–æ–∂–Ω–æ—Å—Ç—å:** –°–∞–º—ã–π –≤—ã—Å–æ–∫–∏–π –ø–æ—Ä–æ–≥ –≤—Ö–æ–∂–¥–µ–Ω–∏—è. –¢—Ä–µ–±—É–µ—Ç –≥–ª—É–±–æ–∫–æ–≥–æ –ø–æ–Ω–∏–º–∞–Ω–∏—è —Å–≤–æ–µ–π –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–π —Ä–∞–±–æ—Ç—ã, —á—Ç–æ–±—ã –∏–∑–±–µ–∂–∞—Ç—å –ø—Ä–æ–±–ª–µ–º —Å –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å—é.

#### **2. SQLAlchemy (Python)**

*   **–§–∏–ª–æ—Å–æ—Ñ–∏—è:** "SQL - —ç—Ç–æ –º–æ—â–Ω–µ–π—à–∏–π —è–∑—ã–∫, —Å–æ–∑–¥–∞–Ω–Ω—ã–π –¥–µ—Å—è—Ç–∏–ª–µ—Ç–∏—è–º–∏. –ú—ã –Ω–µ –±—É–¥–µ–º –µ–≥–æ –ø—Ä—è—Ç–∞—Ç—å, –∞ –¥–∞–¥–∏–º –≤–∞–º –ª—É—á—à–∏–µ –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—ã –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –Ω–∏–º –Ω–∞ —Ä–∞–∑–Ω—ã—Ö —É—Ä–æ–≤–Ω—è—Ö –∞–±—Å—Ç—Ä–∞–∫—Ü–∏–∏".
*   **–ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω:** **Data Mapper**.
*   **–£—Ä–æ–≤–µ–Ω—å –∞–±—Å—Ç—Ä–∞–∫—Ü–∏–∏:** **–ú–Ω–æ–≥–æ—É—Ä–æ–≤–Ω–µ–≤—ã–π**. –í —ç—Ç–æ–º –µ–µ –≥–µ–Ω–∏–∞–ª—å–Ω–æ—Å—Ç—å.
    *   **SQLAlchemy Core:** "–õ–µ–≥–∫–∏–π" —É—Ä–æ–≤–µ–Ω—å. –ü–æ–∑–≤–æ–ª—è–µ—Ç —Å—Ç—Ä–æ–∏—Ç—å SQL-–∑–∞–ø—Ä–æ—Å—ã, –∏—Å–ø–æ–ª—å–∑—É—è Python-–∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏, –Ω–æ –≤—ã –≤—Å–µ –µ—â–µ –¥—É–º–∞–µ—Ç–µ –≤ —Ç–µ—Ä–º–∏–Ω–∞—Ö —Ç–∞–±–ª–∏—Ü –∏ –∫–æ–ª–æ–Ω–æ–∫. –≠—Ç–æ –Ω–µ ORM, –∞ —Å–∫–æ—Ä–µ–µ –ø—Ä–æ–¥–≤–∏–Ω—É—Ç—ã–π SQL Expression Language.
    *   **SQLAlchemy ORM:** "–¢—è–∂–µ–ª—ã–π" —É—Ä–æ–≤–µ–Ω—å, –ø–æ—Å—Ç—Ä–æ–µ–Ω–Ω—ã–π –ø–æ–≤–µ—Ä—Ö Core. –ü—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç –ø–æ–ª–Ω–æ—Ü–µ–Ω–Ω—ã–π Data Mapper —Å —Å–µ—Å—Å–∏—è–º–∏, —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ–º —Å–æ—Å—Ç–æ—è–Ω–∏–µ–º, –ª–µ–Ω–∏–≤–æ–π –∑–∞–≥—Ä—É–∑–∫–æ–π –∏ —Ç.–¥., –æ—á–µ–Ω—å –ø–æ—Ö–æ–∂–∏–π –Ω–∞ Hibernate.
*   **–ö–ª—é—á–µ–≤—ã–µ –æ—Ç–ª–∏—á–∏—è:**
    *   **–î–≤–∞ —É—Ä–æ–≤–Ω—è –≤ –æ–¥–Ω–æ–π –±–∏–±–ª–∏–æ—Ç–µ–∫–µ:** –í—ã –º–æ–∂–µ—Ç–µ –Ω–∞—á–∞—Ç—å —Å –ø—Ä–æ—Å—Ç–æ–≥–æ `Core` –¥–ª—è –±—ã—Å—Ç—Ä—ã—Ö –∑–∞–ø—Ä–æ—Å–æ–≤ –∏ –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ –ø–µ—Ä–µ–π—Ç–∏ –∫ `ORM` –¥–ª—è —Å–ª–æ–∂–Ω—ã—Ö –¥–æ–º–µ–Ω–Ω—ã—Ö –º–æ–¥–µ–ª–µ–π. –ò–ª–∏ –¥–∞–∂–µ —Å–º–µ—à–∏–≤–∞—Ç—å –∏—Ö –≤ –æ–¥–Ω–æ–º –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–∏.
    *   **–Ø–≤–Ω–æ–µ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Å–µ—Å—Å–∏–µ–π:** –í –æ—Ç–ª–∏—á–∏–µ –æ—Ç "–º–∞–≥–∏–∏" –º–Ω–æ–≥–∏—Ö –¥—Ä—É–≥–∏—Ö —Ñ—Ä–µ–π–º–≤–æ—Ä–∫–æ–≤, SQLAlchemy –∑–∞—Å—Ç–∞–≤–ª—è–µ—Ç –≤–∞—Å —è–≤–Ω–æ —É–ø—Ä–∞–≤–ª—è—Ç—å –∂–∏–∑–Ω–µ–Ω–Ω—ã–º —Ü–∏–∫–ª–æ–º —Å–µ—Å—Å–∏–∏ (`session.add()`, `session.commit()`, `session.rollback()`), —á—Ç–æ –¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ –ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã–º.

#### **3. Entity Framework (EF) (.NET)**

*   **–§–∏–ª–æ—Å–æ—Ñ–∏—è:** "–ü—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç—å –≥–ª—É–±–æ–∫—É—é –∏–Ω—Ç–µ–≥—Ä–∞—Ü–∏—é —Å —è–∑—ã–∫–æ–º C# –∏ –ø–ª–∞—Ç—Ñ–æ—Ä–º–æ–π .NET –¥–ª—è –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ –µ—Å—Ç–µ—Å—Ç–≤–µ–Ω–Ω–æ–π —Ä–∞–±–æ—Ç—ã —Å –¥–∞–Ω–Ω—ã–º–∏ –∫–∞–∫ —Å –∫–æ–ª–ª–µ–∫—Ü–∏—è–º–∏ –æ–±—ä–µ–∫—Ç–æ–≤".
*   **–ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω:** **Data Mapper** (—Å `DbContext`, –∫–æ—Ç–æ—Ä—ã–π —Ä–µ–∞–ª–∏–∑—É–µ—Ç `Unit of Work` –∏ `Identity Map`).
*   **–£—Ä–æ–≤–µ–Ω—å –∞–±—Å—Ç—Ä–∞–∫—Ü–∏–∏:** **–í—ã—Å–æ–∫–∏–π ("—Ç—è–∂–µ–ª–∞—è" ORM)**.
*   **–ü–æ–¥—Ö–æ–¥ –∫ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∑–∞–ø—Ä–æ—Å–æ–≤:** **LINQ (Language-Integrated Query)**. –≠—Ç–æ –µ–≥–æ –≥–ª–∞–≤–Ω–æ–µ –æ—Ç–ª–∏—á–∏–µ. –ó–∞–ø—Ä–æ—Å—ã –ø–∏—à—É—Ç—Å—è –∫–∞–∫ —á–∞—Å—Ç—å —è–∑—ã–∫–∞ C#, —á—Ç–æ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç —Å—Ç–∞—Ç–∏—á–µ—Å–∫—É—é —Ç–∏–ø–∏–∑–∞—Ü–∏—é –∏ –ø—Ä–æ–≤–µ—Ä–∫—É –Ω–∞ —ç—Ç–∞–ø–µ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏.
    ```csharp
    // –≠—Ç–æ C#, –∞ –Ω–µ —Å—Ç—Ä–æ–∫–∞. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä –ø—Ä–æ–≤–µ—Ä–∏—Ç, —á—Ç–æ —É User –µ—Å—Ç—å —Å–≤–æ–π—Å—Ç–≤–æ Name.

                       .Where(u => u.City == "London")
                       .OrderBy(u => u.Name)
                       .ToList();
    ```
*   **–ö–ª—é—á–µ–≤—ã–µ –æ—Ç–ª–∏—á–∏—è:**
    *   **LINQ:** –ì–ª–∞–≤–Ω–∞—è "—Ñ–∏—à–∫–∞". –ü–æ–∑–≤–æ–ª—è–µ—Ç –ø–∏—Å–∞—Ç—å –∑–∞–ø—Ä–æ—Å—ã –≤ –æ–±—ä–µ–∫—Ç–Ω–æ-–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–º —Å—Ç–∏–ª–µ —Å –∞–≤—Ç–æ–¥–æ–ø–æ–ª–Ω–µ–Ω–∏–µ–º –∏ –ø—Ä–æ–≤–µ—Ä–∫–æ–π —Ç–∏–ø–æ–≤ –≤ IDE.
    *   **–ú–∏–≥—Ä–∞—Ü–∏–∏:** EF –∏–º–µ–µ—Ç –æ–¥–Ω—É –∏–∑ –ª—É—á—à–∏—Ö –∏ —Å–∞–º—ã—Ö –∑—Ä–µ–ª—ã—Ö —Å–∏—Å—Ç–µ–º –¥–ª—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–π –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∏ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –º–∏–≥—Ä–∞—Ü–∏–π —Å—Ö–µ–º—ã –±–∞–∑—ã –¥–∞–Ω–Ω—ã—Ö.
    *   **–û—Ç—Å–ª–µ–∂–∏–≤–∞–Ω–∏–µ –∏–∑–º–µ–Ω–µ–Ω–∏–π (Change Tracking):** EF –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –æ—Ç—Å–ª–µ–∂–∏–≤–∞–µ—Ç –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –∑–∞–≥—Ä—É–∂–µ–Ω–Ω—ã—Ö –æ–±—ä–µ–∫—Ç–∞—Ö. –í—ã –ø—Ä–æ—Å—Ç–æ –º–µ–Ω—è–µ—Ç–µ —Å–≤–æ–π—Å—Ç–≤–æ `user.Name = "New"`, –∞ –ø—Ä–∏ –≤—ã–∑–æ–≤–µ `context.SaveChanges()` EF —Å–∞–º –ø–æ–Ω–∏–º–∞–µ—Ç, —á—Ç–æ –Ω—É–∂–Ω–æ —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞—Ç—å `UPDATE`.
    *   **–î–≤–∞ –ø–æ–¥—Ö–æ–¥–∞ –∫ —Ä–∞–∑—Ä–∞–±–æ—Ç–∫–µ:** `Code-First` (–≤—ã –ø–∏—à–µ—Ç–µ –∫–ª–∞—Å—Å—ã, –∞ EF —Å–æ–∑–¥–∞–µ—Ç –±–∞–∑—É) –∏ `Database-First` (–≤—ã –∏–º–µ–µ—Ç–µ –±–∞–∑—É, –∞ EF –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –∫–ª–∞—Å—Å—ã).

#### **4. Dapper (.NET)**

*   **–§–∏–ª–æ—Å–æ—Ñ–∏—è:** "ORM - —ç—Ç–æ –∑–ª–æ. `SELECT N+1` - —ç—Ç–æ –∑–ª–æ. –ï–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ–µ, —á—Ç–æ –Ω–∞–º –Ω—É–∂–Ω–æ, - —ç—Ç–æ –æ—á–µ–Ω—å –±—ã—Å—Ç—Ä—ã–π —Å–ø–æ—Å–æ–± –ø—Ä–µ–≤—Ä–∞—Ç–∏—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—ã—Ä–æ–≥–æ SQL-–∑–∞–ø—Ä–æ—Å–∞ –≤ —Å–ø–∏—Å–æ–∫ –æ–±—ä–µ–∫—Ç–æ–≤".
*   **–ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–Ω—ã–π –ø–∞—Ç—Ç–µ—Ä–Ω:** **–ù–µ —è–≤–ª—è–µ—Ç—Å—è ORM –≤ –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–æ–º —Å–º—ã—Å–ª–µ**. –≠—Ç–æ **Micro-ORM** –∏–ª–∏ –ø—Ä–æ—Å—Ç–æ **Object Mapper**.
*   **–£—Ä–æ–≤–µ–Ω—å –∞–±—Å—Ç—Ä–∞–∫—Ü–∏–∏:** **–û—á–µ–Ω—å –Ω–∏–∑–∫–∏–π**.
